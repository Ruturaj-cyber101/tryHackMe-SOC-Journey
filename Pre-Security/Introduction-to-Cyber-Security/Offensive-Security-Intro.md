# Room: Offensive Security Intro

## Overview
This room introduced the concept of offensive security, which focuses on finding vulnerabilities in systems before attackers do.

## What I Learned
- What offensive security means
- Role of ethical hackers and penetration testers
- Difference between ethical hacking and malicious hacking
- Importance of testing system security proactively

## Tools / Concepts
- Ethical hacking
- Penetration testing basics

## Key Takeaway
Offensive security helps organizations discover and fix weaknesses before real attackers exploit them.
